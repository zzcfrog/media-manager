# PRD — 视频分析器

## 1. 产品概述

视频分析器是一款桌面端媒体管理工具，核心能力是通过 AI 对视频和图片进行结构化分析，将非结构化的视觉内容转化为可搜索、可组织的结构化数据。

目标用户：摄影师、视频创作者、媒体资产管理者。

## 2. 支持的媒体格式

### 视频

`.mp4` `.mov` `.avi` `.mkv` `.webm` `.m4v` `.flv` `.wmv` `.3gp` `.mts` `.m2ts`

### 图片

覆盖常见格式和主流相机 RAW：
- 常见：JPG、PNG、WebP、BMP、TIFF
- Apple/现代：HEIC、HEIF、AVIF
- Canon：CR2、CR3
- Nikon：NEF、NRW
- Sony：ARW、SRF、SR2
- Fujifilm：RAF
- Olympus：ORF
- Panasonic：RW2
- Hasselblad：3FR、FFF
- Phase One：IIQ
- Pentax：PEF
- Samsung：SRW
- 其他：DNG、RAW、MEF、MOS、KDC、DCR、MRW、ProRAW

## 3. 核心功能

### 3.1 媒体导入

**入口**：导入对话框（拖拽 / 选择文件 / 选择目录）

**流程**：
1. 选择文件或目录路径
2. 系统扫描路径下所有匹配格式的文件，跳过已导入的（按文件完整路径去重）
3. 并发导入（3 个同时），每个文件：
   - 提取元数据（视频用 ffprobe + exiftool，图片用 exiftool）
   - 生成缩略图（ffmpeg 截帧或 exiftool 提取内嵌缩略图）
   - 写入数据库
4. 展示结果：成功数、跳过数（重复）、失败数

**元数据提取**：
- 视频：分辨率、时长、帧率、视频编码、码率、音频编码/采样率/声道、色彩空间、相机制造商/机型/镜头、拍摄日期
- 图片：分辨率、相机制造商/机型/镜头、拍摄日期、色彩空间、位深度

### 3.2 媒体浏览

**视图模式**：
- **网格视图**：自适应列数，卡片宽度可调（0.5x ~ 2x），支持按时间线分组
- **列表视图**：表头可排序，信息密度更高

**筛选**：
- 媒体类型（全部/图片/视频），按钮组带 tooltip 提示
- 星级评分（1-5 星）
- 颜色标签（红/黄/绿/蓝/紫）
- 仅收藏
- 全文搜索（文件名 + AI 分析内容）
- 按文件夹筛选（侧边栏素材库独立目录树，点击筛选）

**排序**：导入时间、拍摄时间、文件名、分辨率、时长、文件大小、评分

**分组**：按天/周/月/季/年、按时长分段

**无限滚动**：每页 60 条，自动预加载。

### 3.3 AI 分析

**视频分析**（每个分段 16 个维度）：

| 维度 | 说明 | 取值 |
|------|------|------|
| time_start / time_end | 分段时间范围 | MM:SS.ss |
| visual | 画面内容描述 | 自由文本 |
| asr | 语音内容 | 由独立 ASR 引擎提供 |
| subtitle | 画面字幕 | 自由文本 |
| dominant_colors | 主要颜色 | 字符串数组 |
| main_subjects | 主要主体 | 字符串数组 |
| shot_type | 景别 | 全/远/中/近/特 |
| focal_length | 焦段 | 广角/中焦/长焦 |
| camera_angle | 视角 | 俯视/仰视/平视/上帝视角 |
| camera_movement | 运镜 | 推/拉/摇/移/绕/固定 |
| perspective | 透视 | 主观/客观/旁听 |
| scene_type | 场景类型 | 室内/自然/城市/水下 |
| mood | 情绪氛围 | 欢快/忧伤/紧张/平静/浪漫 |
| lighting | 光线 | 自然光/逆光/侧光 |
| weather | 天气 | 晴天/阴天/雨/雪/夜景 |

**图片分析**（13 个维度）：与视频类似，额外包含构图（三分法/对称/引导线/框架/中心/对角线）和风格（纪实/艺术/商业/建筑/风光/人像/街拍）。

**分析流程**：

全局设置（存储在数据库）控制分析参数，点击"AI 分析"时弹出确认窗显示当前设置摘要和费用提示，确认后开始分析。

- **多模态模式**（默认）：VLM 同时处理视觉和语音，3 阶段流程：
  1. 视频压缩（按全局设置选择分辨率 480P/320P/240P，帧率 30/20/10）
  2. 视频编码
  3. AI 音视频分析（VLM 同时识别画面和语音）

- **独立 ASR 模式**：VLM 处理视觉 + 本地 ASR 引擎并行处理语音，4 阶段流程：
  1. 视频压缩
  2. 视频编码
  3. AI 模型分析（VLM）+ 语音识别（ASR）并行执行
  4. ASR 结果按时间戳合并到 VLM 分段中
  5. 保存到数据库，建立全文搜索索引

**支持的模型**：
- 智谱 AI：GLM-4V-Plus（旗舰）、GLM-4.6V（性价比）、GLM-4.6V-Flash（免费）、GLM-4.6V-FlashX、GLM-4.5V
- OpenAI GPT-4o（即将支持）
- Google Gemini 2.5 Pro（即将支持）

**全局设置项**：
- 压缩分辨率（480P/320P/240P）
- 帧率（30/20/10 fps）
- AI 模型选择
- 视频 API Key（留空使用系统环境变量）
- 多模态解析开关（开启时 VLM 同时处理音频，关闭时使用独立 ASR 引擎）
- ASR 引擎选择（目前支持本地 Whisper，预留云端扩展）
- ASR API Key（云端 ASR 引擎时使用，本地模型可留空）

### 3.4 搜索

基于 FTS5 全文搜索，覆盖：
- 文件名
- AI 分析的视觉描述、语音内容、字幕
- 主要主体、主要颜色

中文搜索使用 jieba 分词。

### 3.5 收藏管理

- 星级评分：1-5 星
- 颜色标签：红、黄、绿、蓝、紫
- 喜欢标记：是/否（红心图标）

### 3.6 合集

- 创建命名合集
- 添加媒体到合集（自动设封面为第一个添加的项）
- 浏览合集内容
- 删除合集

### 3.7 详情页

**媒体查看**：
- 视频：HTML5 播放器 + 音频波形 + 三种专业示波器（Waveform / Parade / Vectorscope）
- 图片：居中显示 + RGB 直方图

**分析结果浏览与编辑**：
- 分段列表，点击跳转到对应时间点
- 播放时自动高亮并滚动到当前分段
- **所有字段可编辑**：点击文本字段进入编辑（contenteditable），失焦自动保存；标签（颜色/主体）点击 × 移除
- 编辑后自动刷新 FTS 搜索索引
- 清除分析 / 重新分析

**元数据展示**：
- 技术信息：分辨率、时长、编码、帧率、码率、音频参数、色彩空间
- 相机信息：制造商、机型、镜头
- 文件信息：路径、大小、导入日期、拍摄日期

### 3.8 重复素材检测

**入口**：侧边栏"查找重复"菜单项，进入独立重复页。

**四层检测**：
- **重复**：基于文件内容 SHA256 哈希，检测不同路径下的相同文件
- **酷似**：ResNet50 特征余弦相似度 ≥ 96%，如同张照片的 JPG 和 RAW
- **相似**：ResNet50 特征余弦相似度 ≥ 90%，如连拍照片
- **聚类**：HDBSCAN 密度聚类，同场景不同角度或时间

仅图片参与相似检测，视频排除。

**导入时自动计算**：
- 所有媒体自动计算 file_hash（SHA256 分块读取）
- 图片额外计算 ResNet50 特征向量（2048 维 L2 归一化），存为 8KB BLOB
- 已有素材可通过"计算特征向量"按钮批量补算

**展示方式**：
- 四个标签页切换（重复/酷似/相似/聚类），右上角问号图标显示说明
- 按组展示，每组显示缩略图横排 + 文件名 + 文件大小
- 相似/酷似模式显示相似度百分比
- 单击选中，双击进入详情页
- 支持多选（Ctrl/Shift + 点击）、框选（鼠标拖拽）
- 右键菜单支持"查看详情"、"在文件夹中显示"、"移出素材库"
- 键盘快捷键：方向键移动选中、Enter 打开详情、Backspace 返回画廊

### 3.9 键盘快捷键

全局快捷键系统，覆盖 Gallery 和 Detail 页面。顶部工具栏设置图标左侧有键盘按钮，点击弹出快捷键参考弹窗。

**Gallery 快捷键**：方向键移动选中、Enter 打开详情、Delete 删除、1-5 评分、F 切换喜欢、G 视图切换、`/` 搜索聚焦。

**Detail 快捷键**：←→ 切换上/下一个素材、1-5 评分、F 切换喜欢、Space 播放/暂停、Backspace 返回画廊。编辑状态时快捷键自动禁用。

## 4. 用户工作流

```
导入媒体 → 浏览筛选 → AI 分析 → 查看结果 → 组织管理
                                    ↓
                        按分段浏览，点击跳转
                        阅读视觉描述/语音/字幕
                        查看镜头和场景维度
```

### 3.10 文件夹浏览

Lightroom 风格的目录树导航，侧边栏"素材库"菜单右侧有独立展开箭头控制文件树的显示/隐藏。

- **目录树**：根据导入媒体的 `file_path` 自动构建，使用 `q-tree` 组件展示
- **媒体计数**：每个文件夹节点显示其及子文件夹下的媒体总数
- **点击筛选**：点击文件夹节点，画廊仅显示该文件夹及其子文件夹下的媒体
- **再次点击取消**：点击已选中的文件夹可取消筛选，恢复显示全部
- **热区分离**：点击"素材库"文字/图标导航到画廊首页，点击右侧箭头展开/折叠文件树
- **详情页联动**：进入详情页时自动展开文件树并定位到该文件所在目录
- **长名称截断**：文件夹名称过长时 ellipsis 截断，hover 显示 tooltip 全名
- **无额外存储**：目录结构从现有数据动态生成，无需额外数据库表或 schema 变更

## 5. 当前限制

| 限制 | 说明 |
|------|------|
| 单视频分析 | 不支持批量分析，每次只能分析一个 |
| 单用户 | 无用户认证和多用户支持 |
| 仅删除库记录 | 删除媒体只移除数据库记录，不删除源文件 |
| OpenAI/Google 未支持 | 模型选择中标记"即将支持" |
| 无导出功能 | 分析结果不能导出为文件 |
| 无 GPS 数据 | 不提取或显示地理位置信息 |
| ASR 为本地模型 | 使用 faster-whisper 占用 3-6GB 内存，后续计划改为云端 |
| 桌面端为主 | 无移动端适配 |
