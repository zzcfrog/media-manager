# UE Design — 视频分析器

## 整体风格

- **桌面优先**的 Electron 应用，无移动端适配
- **暗色主题**为默认，支持亮色和跟随系统
- 8 种可选主题色（accent），默认蓝紫 `#6c8cff`
- Quasar 组件库，圆角卡片（16px），毛玻璃对话框（backdrop-filter: blur）
- Lottie 动画用于 AI 分析的等待状态

## 主题系统

三种模式：暗色 → 亮色 → 跟随系统。通过顶部工具栏调色板按钮切换。

| 变量 | 暗色 | 亮色 |
|------|------|------|
| `--bg` | `#0a0a0a` | `#f5f5f5` |
| `--surface` | `#141414` | `#fff` |
| `--text` | `#e8e8e8` | `#222` |
| `--accent` | 用户可选 | 用户可选 |

可选主题色：蓝紫、浅蓝、青色、绿色、琥珀、深橙、红色、紫色。

Hash 路由，无 Vue Router：

## Gallery 页面

### 筛选栏

顶部水平排列，flex-wrap 自动换行：

| 控件 | 说明 |
|------|------|
| 类型切换 | 全部 / 图片 / 视频（`q-btn-group` + 独立 `q-btn`，各带 `q-tooltip` 1s 延迟） |
| 星级筛选 | 1-5 星，点击切换 |
| 颜色标签 | 5 色圆点（红黄绿蓝紫），点击切换 |
| 收藏 | 三态：全部 → 已喜欢 → 未喜欢（CSS 伪元素斜杠标识"排除"），循环切换 |
| 已分析 | 三态：全部 → 已分析 → 未分析（CSS 伪元素斜杠标识"排除"），循环切换 |
| 重置 | 有激活筛选时显示，重置所有筛选和搜索条件 |
| 搜索框 | 全文搜索（文件名 + 分析内容） |
| 排序 | 下拉选择 + 升降序切换 |
| 分组 | 按时间/时长分组（排序字段为时间时可用） |
| 视图 | 网格 / 列表 切换 |

### 网格视图

CSS Grid 自适应列数，卡片宽度通过底部滑块控制（0.5x ~ 2x）。可按时间线分组显示。**50% 缩放时隐藏卡片文字（文件名、元数据），进入紧凑模式只显示缩略图和叠加信息**。使用 `content-visibility: auto` 优化大列表渲染性能。

**卡片内容**：
- 16:9 缩略图
- 类型图标（视频/图片，右下角）
- XMP 徽章（图片已写入 XMP 时，右下角类型图标左侧，显示自定义 SVG 图标）
- AI 分析徽章（已完成分析时，左下角）
- 喜欢红心
- 颜色圆点
- 星级
- 文件名、分辨率/相机/时长

### 列表视图

表头固定，可排序。每行显示 64x36 缩略图 + 文件名 + 分辨率 + 时长 + 大小 + 拍摄日期 + 导入日期 + "标记"列（评分 + 颜色标签 + 收藏 + 分析状态）。

### 交互

| 操作 | 行为 |
|------|------|
| 单击 | 选中 |
| Ctrl/Cmd+点击 | 切换选中 |
| Shift+点击 | 范围选中 |
| 双击 | 打开详情 |
| 右键 | 自定义上下文菜单（查看详情、移除、查找相似、AI 分析） |
| 拖拽框选 | Lasso 选择（自动滚动边缘） |
| Ctrl+Delete | 删除选中 |
| Enter | 打开选中项详情 |

### 导入流程

三步弹窗：

1. **选择**：拖拽区域 + "选择文件" / "选择目录" 按钮（Electron 原生对话框）
2. **进度**：扫描文件 → 3 个并发导入 → 线性进度条 + 已导入缩略图网格
3. **完成**：导入数、跳过数（重复）、失败数 + 可展开详情列表

### 文件夹目录树

侧边栏"素材库"菜单行右侧有独立展开箭头按钮，点击展开/折叠目录树。点击"素材库"文字/图标区域导航到画廊首页。**默认折叠，不自动展开**。

```
▸ 素材库                              [▾]
  │ Photos    120
  │├─ 2025     80
  │└─ 2026     40
  └ Videos     35
   ├─ Travel   20
   └─ Family   15
```

**样式**：
- 无连接线（`no-connectors`），紧凑模式（`dense`）
- **VS Code 风格竖线缩进**：`q-tree__children` 左侧 `border-left: 1px solid var(--border)`，缩进减小约 1/3
- 叶节点保留三角形箭头占位空间（不显示箭头但占位）
- 每个节点：文件夹图标（灰色 `folder`，选中时 accent 色）+ 文件夹名 + 右侧媒体计数（灰色小字）
- 展开状态通过 `:expanded` 属性响应式控制
- 选中节点：背景变 `var(--accent-dim)` + accent 文字；**选中与悬浮高亮区域统一**（均为 `q-tree__node-header` 全宽）
- 长文件夹名 ellipsis 截断 + `q-tooltip` 显示全名
- 抽屉宽度 260px
- 浏览子项（素材库/我的喜欢/查找重复）与"浏览"标题左对齐，文件夹根目录与"素材库"对齐

**交互**：

| 操作 | 行为 |
|------|------|
| 点击"素材库"文字/图标 | 导航到画廊首页，清除筛选 |
| 点击右侧展开箭头 | 展开/折叠目录树 |
| 点击文件夹 | 选中该文件夹，画廊仅显示此文件夹及子文件夹下的媒体 |
| 再次点击已选中文件夹 | 取消选中，恢复显示全部 |
| 进入详情页 | 自动展开目录树并定位到文件所在目录 |

## Detail 页面

### 布局

双栏布局，元数据信息显示在媒体上方：

```
┌──────────────────────────┬─────────────────────┐
│  工具栏（返回、文件名、评分、颜色、收藏、XMP）          │
├──────────────────────────┤                     │
│  元数据横排栏（图像/视频·音频·拍摄设备·文件信息）       │
├──────────────────────────┤                     │
│                          │  分析结果            │
│     视频播放器 / 图片预览  │  （分段列表 /         │
│     （图片支持缩放拖拽）    │   分析进度）          │
│                          │                     │
├──────────────────────────┤                     │
│  音频波形                 │                     │
├──────────────────────────┤                     │
│  Waveform │ Parade │ Vec │                     │
└──────────────────────────┴─────────────────────┘
       flex: 1                     420px 固定
```

- 元数据栏：横排显示多个区块（图像/视频、音频、拍摄设备、文件信息），每个区块带标题和左侧强调色竖线
- 视频元数据栏：视频信息分两列（左列：分辨率/时长/编码，右列：帧率/码率/色彩空间/色彩曲线）
- 色彩曲线：显示 PictureControlName（尼康 N-Log 取自 EXIF，大疆 D-Log M 从文件名推断），带叹号提示准确性说明
- 图片预览：支持缩放（鼠标滚轮/触控板双指张合）和拖拽平移（触控板双指滑动/鼠标拖拽），缩放条贴在图片底边内侧
- **全屏看图**：F 键或点击缩放栏全屏按钮，浏览器 Fullscreen API 全屏显示图片区域，Escape/F 退出
- **导航缩略图（Minimap）**：放大 > 100% 时右下角出现半透明缩略图（160px 宽），蓝色矩形显示当前视口位置，点击可跳转
- 文件路径：完整显示，超长时在文件信息区块内折行

### 工具栏

- 返回按钮 → `#/gallery`
- 文件名标题
- 星级评分（1-5 星，可点击）
- 颜色标签（5 色圆点）
- 喜欢按钮
- XMP 写入按钮（仅图片 + 已分析完成时显示）：未写入时显示写入图标（文件+箭头），已写入时显示刷新图标（文件+循环箭头）

### 元数据面板

横排显示在媒体上方，分为 3-4 个区块（视频4块：视频·音频·拍摄设备·文件信息；图片3块：图像·拍摄设备·文件信息），每块标题带左侧强调色竖线。前三块固定宽度不收缩，最后一块（文件信息）自适应剩余空间。

### 视频播放器

原生 HTML5 `<video>` 控件，下方三个 Canvas：

1. **音频波形**（64px 高）：点击跳转，播放时带动画游标
2. **Waveform Scope**：亮度分布
3. **Parade Scope**：RGB 三通道波形
4. **Vectorscope**：色度圆形分布图

播放时以 ~15fps 实时渲染。图片显示 RGB 直方图。

### 设置弹窗

点击顶部工具栏设置图标打开，**标签页布局**（4 个标签），底部"确定"按钮保存。

**通用**标签页：
- 语言选择：下拉菜单（中文 / English），切换后全页面即时更新
- 主题模式：暗色 / 亮色 / 跟随系统
- 强调色：8 种主题色圆点选择（蓝紫、浅蓝、青色、绿色、琥珀、深橙、红色、紫色）

**图片分析**标签页：
- 压缩尺寸：1920（推荐）/ 1280 / 800 / 原图
- AI 模型：下拉选择（智谱 GLM 系列等，独立于视频模型）
- API Key：密码输入框（独立于视频 API Key，必须设置）

**视频分析**标签页：
- 压缩分辨率：480P（推荐）/ 320P / 240P
- 帧率：30fps（推荐）/ 20fps / 10fps
- 预估码率：根据分辨率/帧率动态显示（480P 30fps = 2.00 Mbps）
- 硬件加速压缩：开关（仅当检测到硬件编码器时显示，macOS VideoToolbox GPU 解码）
- AI 模型：下拉选择（智谱 GLM 系列等）
- API Key：密码输入框，右侧眼睛图标切换显示/隐藏，留空则使用系统环境变量

**音频分析**标签页：
- 开关："使用视频模型进行多模态解析"（默认开启）
- ASR 引擎选择（仅在开关关闭时可用，当前支持本地 Whisper）
- ASR API Key（仅在开关关闭时显示）：密码输入框 + 眼睛图标，云端 ASR 时填写，本地模型可留空

### Footer 筛选条件标签

Gallery footer 中"共 N 个素材"右侧、缩放控件左侧居中显示当前活跃的筛选条件标签：

- 每个条件显示为独立圆角背景标签（`surface2` 色背景）
- 文件夹：📁 图标 + **完整文件夹路径**
- 媒体类型：image/video 图标 + 类型名
- 评分：星星字符（无 icon，如 ★★★）
- 颜色标签：彩色圆点（无文字）
- 已喜欢：favorite 图标 + 文字
- 未喜欢：favorite_border 图标 + 文字 + CSS 斜杠（仅打在图标上）
- 已分析/未分析：auto_awesome 图标 + 文字（未分析同样仅图标有斜杠）
- 搜索：search 图标 + 搜索词
- 无条件时不显示

### 分析进度

时间线样式，阶段数根据音频模式动态变化：

**多模态模式（3 阶段）**：
```
 ● 视频压缩 [====45%====]  ● 视频编码  ● AI 音视频分析
```

**独立 ASR 模式（4 阶段）**：

```
 ● 视频压缩 [====45%====]  ● 视频编码  ● AI 模型分析  ● 语音识别(ASR)
```

- 圆点：等待灰色 → 进行中脉冲动画 + 蓝色 → 完成绿色勾
- 压缩阶段：解析 ffmpeg stderr 推送真实百分比进度条
- 编码阶段：完成时显示压缩后文件大小（自动选择 KB/MB/GB）
- AI 分析阶段子步骤：上传至 AI 服务 → 接收结果 (N 字)
- 独立 ASR 模式：VLM 完成立刻标记，ASR 独立显示"加载语音模型"→"语音识别中"
- 进度条 + 状态文本 + 已用时间
- AI 分析阶段显示打字机效果的随机分析描述（20 条轮播，有子步骤时显示子步骤替代）

### 时间线工具栏

底部面板工具栏左侧新增 **[时间线] [脑图]** 切换按钮（`q-btn-toggle`，accent 高亮选中态）。切换时保持数据不变，仅切换展示方式。

时间线模式下工具栏包含四组功能：

- **播放控制**：跳到开头 / 播放暂停 / 跳到结尾 / 倍速选择（0.5x-2x），联动预览区视频播放器
- **编辑操作**：撤销 / 重做（快照栈，时间线/脑图两视图通用，脑图编辑也可撤销）；删除（需选中轨道项，右键菜单或 Delete 快捷键）
- **缩放**：缩小 / 滑块（1-10x）/ 放大 / 适配宽度，通过 scaleX 放大轨道内容
- **添加轨道**：下拉菜单选择轨道类型（主题/情绪/旁白/字幕/文字/视频）

**行头冻结**：左侧行头（`.wb-track-label`，60px 宽）使用 `position: sticky; left: 0`，横向滚动时间线时六个轨道标签 + 标尺左侧空白固定不动，轨道内容在其后滚动。playhead（z-index 10）始终在行头（z-index 6）之上。

**片段缩放联动**：拖拽调整 video 片段左右边缘时，落在该 video 原时间区间内的 emotion/旁白/字幕/文字块会按比例同步缩放（缩短则等比缩短、拉长则等比拉长），保持各轨道块与所属 video 段对齐。

**非视频块编辑约束**（保持脑图与时间线一致）：情绪/旁白/字幕/文字/主旨等非视频块**不能手动拖边缘改时长**（鼠标在边缘不显示缩放光标、不进入缩放，时长随所属 video 片段被动联动）。**删除主旨或叙述块**会弹窗确认，确认后连同其下所有叙述/镜头一并删除。

**主旨/叙事 overlay 框**（对齐脑图层级）：时间线不是 6 条平行轨道，而是嵌套两层——
- **情绪轨道**独立在最上（情绪曲线 + 点），**不被框覆盖**（情绪是分镜元数据，不分组）。
- **内容组**（情绪下方）：旁白/字幕/分镜 3 条贯穿平行轨道。
- **主旨框**（覆盖框）：横跨整个主旨时长、纵向包住 3 条内容轨道，彩色实线边框 + 标题。
- **叙事框**：在主旨框内、横跨单个叙事时长，虚线边框 + 标题。

点框标题选中框、双击改名、Delete 删除（级联其下所有内容）；点框内分镜/旁白/字幕块正常编辑（框不挡）。

脑图模式下，底部面板替换为垂直滚动的三级层次结构：

```
┌─ mm-act (彩色左边框) ───────────────────── [n个镜头] [🗑] ────────┐

│  │ ▼ 叙事文字：这组镜头的叙事文字...                               ││

│  ～～～ 情绪曲线 sparkline ～～～                                │
└─────────────────────────────────────────────────────────────────┘
```

**交互**：
- **双击编辑**：标题、purpose、叙事文字、旁白等文本字段双击进入编辑模式
- **拖拽排序**：同层级拖拽调整顺序，跨层级拖拽移动（Shot 可拖到其他 Narrative 下）
- **删除**：每个层级 hover 时显示删除按钮，Act 和 Narrative 需确认
- **点击 Shot**：定位视频播放器到对应时间线位置
- **折叠/展开**：Act 和 Narrative 可折叠，默认全部展开
- **与时间线双向同步**：脑图编辑保存后重新 apply 刷新 tracks；反过来，时间线里缩放片段、改旁白/情绪、重排或删除 video 片段后，对应 shot 的时长/内容会回写到脑图。重排为位置驱动——video 片段落在哪个叙事的时间区间，shot 就归到那个叙事，两端叙事时长随之增减

**Shot 拖拽动画细节**：
- **被拖卡片**：拖起后缩小变虚影（opacity 0.3 + scale 0.9），标识该位置已被移出
- **FLIP 重排**：拖动时所有卡片（含虚影）实时移动到"假设松手后"的目标位置 —— 如把 ABCDE 中的 B 拖向末尾，CDE 立即左移填满 B 的空位、B 虚影移到末尾，整个过程丝滑动画
- **no-op 不显示空隙**：在同一叙事内拖动但未跨越其他分镜时（插入位置等于原位置），不出现空隙和虚线框，因为没有实际移动
- **无缝释放**：松手时因拖拽过程中卡片已就位，释放瞬间无多余回弹动画，虚影平滑变回实体
- **跨叙事/外部拖入**：显示带缩略图的虚线占位框（dashed border），兄弟卡片让出空隙
- **拖拽时隐藏 hover 详情**：拖动期间不显示分镜悬停详情弹窗，避免遮挡目标位置

| 悬浮段落 | 右侧显示编辑/删除/AI 重新生成按钮 |
| 点击 shot | 选中时间线上对应的视频线条目 + 预览区播放该片段 |
| 拖拽 shot 到其他段落 | 跨段落移动，自动更新时间范围 |

**视觉设计**：
- 段落左侧有 4px accent 色竖线（类似元数据面板的强调色竖线）
- 当前播放位置对应的段落自动高亮
- 选中段落背景 `var(--accent-dim)`

## 通用组件

| 组件 | 实现方式 |
|------|---------|
| 通知 | `Quasar.Notify`，顶部弹出，1800ms 自动消失 |
| 对话框 | `q-dialog` + `q-card`，16px 圆角，毛玻璃背景。下拉菜单覆盖方角样式 `.q-menu--square { border-radius: 8px }` |
| 上下文菜单 | 自定义 HTML 定位，含 `q-list` + 快捷键提示 |
| 颜色选择 | 行内圆点，点击切换（非独立组件） |
| 进度条 | 自定义 div（`var(--accent)` 色）+ "done/total" 标签 + 24px Lottie 动画 + 弹出任务列表 |
| 动画 | Lottie（AI 分析等待） |
| 无限滚动 | IntersectionObserver，200px 预加载距离。小缩放时 `_checkFill()` 自动检测内容是否填满容器，不足时自动加载更多 |
