# UE Design — 视频分析器

## 整体风格

- **桌面优先**的 Electron 应用，无移动端适配
- **暗色主题**为默认，支持亮色和跟随系统
- 8 种可选主题色（accent），默认蓝紫 `#6c8cff`
- Quasar 组件库，圆角卡片（16px），毛玻璃对话框（backdrop-filter: blur）
- Lottie 动画用于 AI 分析的等待状态

## 主题系统

三种模式：暗色 → 亮色 → 跟随系统。通过顶部工具栏调色板按钮切换。

| 变量 | 暗色 | 亮色 |
|------|------|------|
| `--bg` | `#0a0a0a` | `#f5f5f5` |
| `--surface` | `#141414` | `#fff` |
| `--text` | `#e8e8e8` | `#222` |
| `--accent` | 用户可选 | 用户可选 |

可选主题色：蓝紫、浅蓝、青色、绿色、琥珀、深橙、红色、紫色。

## 页面结构

### 导航

Hash 路由，无 Vue Router：
- `#/gallery` — 媒体库（默认）
- `#/favorites` — 收藏筛选
- `#/collections/:id` — 合集筛选
- `#/detail/:id` — 媒体详情

Gallery 使用 `<keep-alive>` 缓存状态，Detail 每次进入重建。

### 页面布局

```
┌──────────────────────────────────────────────────┐
│  Header（标题 + 主题切换）                         │
├──────────┬───────────────────────────────────────┤
│          │                                       │
│  左侧栏   │              主内容区                   │
│  (260px  │                                       │
│  素材库   │                                       │
│  ├目录树  │                                       │
│  收藏夹) │                                       │
│          │                                       │
├──────────┴───────────────────────────────────────┤
│  Footer（统计信息 + 缩略图大小滑块）                 │
└──────────────────────────────────────────────────┘
```

左侧栏（260px 宽，从上到下）：素材库（右侧独立箭头控制展开/折叠，内嵌文件夹目录树）→ 收藏夹列表，可折叠。

## Gallery 页面

### 筛选栏

顶部水平排列，flex-wrap 自动换行：

| 控件 | 说明 |
|------|------|
| 类型切换 | 全部 / 图片 / 视频（`q-btn-group` + 独立 `q-btn`，各带 `q-tooltip` 1s 延迟） |
| 星级筛选 | 1-5 星，点击切换 |
| 颜色标签 | 5 色圆点（红黄绿蓝紫），点击切换 |
| 收藏 | 仅显示喜欢的素材 |
| 重置 | 有激活筛选时显示 |
| 搜索框 | 全文搜索（文件名 + 分析内容） |
| 排序 | 下拉选择 + 升降序切换 |
| 分组 | 按时间/时长分组（排序字段为时间时可用） |
| 视图 | 网格 / 列表 切换 |

### 网格视图

CSS Grid 自适应列数，卡片宽度通过底部滑块控制（0.5x ~ 2x）。可按时间线分组显示。

**卡片内容**：
- 16:9 缩略图
- 类型图标（视频/图片，右下角）
- XMP 徽章（图片已写入 XMP 时，右下角类型图标左侧，显示自定义 SVG 图标）
- AI 分析徽章（已完成分析时，左下角）
- 喜欢红心
- 颜色圆点
- 星级
- 文件名、分辨率/相机/时长

### 列表视图

表头固定，可排序。每行显示 64x36 缩略图 + 文件名 + 分辨率 + 时长 + 大小 + 拍摄日期 + 导入日期 + 评分/收藏/颜色/分析状态。

### 交互

| 操作 | 行为 |
|------|------|
| 单击 | 选中 |
| Ctrl/Cmd+点击 | 切换选中 |
| Shift+点击 | 范围选中 |
| 双击 | 打开详情 |
| 右键 | 自定义上下文菜单（查看详情、移除） |
| 拖拽框选 | Lasso 选择（自动滚动边缘） |
| Ctrl+Delete | 删除选中 |
| Enter | 打开选中项详情 |

### 导入流程

三步弹窗：

1. **选择**：拖拽区域 + "选择文件" / "选择目录" 按钮（Electron 原生对话框）
2. **进度**：扫描文件 → 3 个并发导入 → 线性进度条 + 已导入缩略图网格
3. **完成**：导入数、跳过数（重复）、失败数 + 可展开详情列表

### 文件夹目录树

侧边栏"素材库"菜单行右侧有独立展开箭头按钮，点击展开/折叠目录树。点击"素材库"文字/图标区域导航到画廊首页。

```
▸ 素材库                              [▾]
  ├─ 📁 Photos    120
  │   ├─ 📁 2025   80
  │   └─ 📁 2026   40
  └─ 📁 Videos     35
      ├─ 📁 Travel  20
      └─ 📁 Family  15
```

**样式**：
- 无连接线（`no-connectors`），紧凑模式（`dense`）
- 每个节点：文件夹图标（灰色 `folder`）+ 文件夹名 + 右侧媒体计数（灰色小字）
- 展开状态通过 `:expanded` 属性响应式控制
- 选中节点：背景变 `var(--accent-dim)` + 白色文字
- 长文件夹名 ellipsis 截断 + `q-tooltip` 显示全名
- 抽屉宽度 260px

**交互**：

| 操作 | 行为 |
|------|------|
| 点击"素材库"文字/图标 | 导航到画廊首页，清除筛选 |
| 点击右侧展开箭头 | 展开/折叠目录树 |
| 点击文件夹 | 选中该文件夹，画廊仅显示此文件夹及子文件夹下的媒体 |
| 再次点击已选中文件夹 | 取消选中，恢复显示全部 |
| 进入详情页 | 自动展开目录树并定位到文件所在目录 |

## Detail 页面

### 布局

双栏布局，元数据信息显示在媒体上方：

```
┌──────────────────────────┬─────────────────────┐
│  工具栏（返回、文件名、评分、颜色、收藏、XMP）          │
├──────────────────────────┤                     │
│  元数据横排栏（图像/视频·音频·拍摄设备·文件信息）       │
├──────────────────────────┤                     │
│                          │  分析结果            │
│     视频播放器 / 图片预览  │  （分段列表 /         │
│     （图片支持缩放拖拽）    │   分析进度）          │
│                          │                     │
├──────────────────────────┤                     │
│  音频波形                 │                     │
├──────────────────────────┤                     │
│  Waveform │ Parade │ Vec │                     │
└──────────────────────────┴─────────────────────┘
       flex: 1                     420px 固定
```

- 元数据栏：横排显示多个区块（图像/视频、音频、拍摄设备、文件信息），每个区块带标题和左侧强调色竖线
- 视频元数据栏：视频信息分两列（左列：分辨率/时长/编码，右列：帧率/码率/色彩空间/色彩曲线）
- 色彩曲线：显示 PictureControlName（尼康 N-Log 取自 EXIF，大疆 D-Log M 从文件名推断），带叹号提示准确性说明
- 图片预览：支持缩放（鼠标滚轮/触控板双指张合）和拖拽平移（触控板双指滑动/鼠标拖拽），缩放条贴在图片底边内侧
- **全屏看图**：F 键或点击缩放栏全屏按钮，浏览器 Fullscreen API 全屏显示图片区域，Escape/F 退出
- **导航缩略图（Minimap）**：放大 > 100% 时右下角出现半透明缩略图（160px 宽），蓝色矩形显示当前视口位置，点击可跳转
- 文件路径：完整显示，超长时在文件信息区块内折行

### 工具栏

- 返回按钮 → `#/gallery`
- 文件名标题
- 星级评分（1-5 星，可点击）
- 颜色标签（5 色圆点）
- 喜欢按钮
- XMP 写入按钮（仅图片 + 已分析完成时显示）：未写入时显示写入图标（文件+箭头），已写入时显示刷新图标（文件+循环箭头）

### 元数据面板

横排显示在媒体上方，分为 3-4 个区块（视频4块：视频·音频·拍摄设备·文件信息；图片3块：图像·拍摄设备·文件信息），每块标题带左侧强调色竖线。前三块固定宽度不收缩，最后一块（文件信息）自适应剩余空间。

### 视频播放器

原生 HTML5 `<video>` 控件，下方三个 Canvas：

1. **音频波形**（64px 高）：点击跳转，播放时带动画游标
2. **Waveform Scope**：亮度分布
3. **Parade Scope**：RGB 三通道波形
4. **Vectorscope**：色度圆形分布图

播放时以 ~15fps 实时渲染。图片显示 RGB 直方图。

### 设置弹窗

点击顶部工具栏设置图标打开，三个区域（图片分析 → 视频分析 → 音频分析），分区标题使用左侧强调条（`border-left: 3px solid var(--accent)`），底部"确定"按钮保存。

**图片分析**：
- 压缩尺寸：1920（推荐）/ 1280 / 800 / 原图
- AI 模型：下拉选择（智谱 GLM 系列等，独立于视频模型）
- API Key：密码输入框（独立于视频 API Key，必须设置）

**视频分析**：
- 压缩分辨率：480P（推荐）/ 320P / 240P
- 帧率：30fps（推荐）/ 20fps / 10fps
- 预估码率：根据分辨率/帧率动态显示（480P 30fps = 2.00 Mbps）
- 硬件加速压缩：开关（仅当检测到硬件编码器时显示，macOS VideoToolbox GPU 解码）
- AI 模型：下拉选择（智谱 GLM 系列等）
- API Key：密码输入框，右侧眼睛图标切换显示/隐藏，留空则使用系统环境变量

**音频分析**：
- 开关："使用视频模型进行多模态解析"（默认开启）
- ASR 引擎选择（仅在开关关闭时可用，当前支持本地 Whisper）
- ASR API Key（仅在开关关闭时显示）：密码输入框 + 眼睛图标，云端 ASR 时填写，本地模型可留空

### AI 分析确认弹窗

点击"AI 分析"按钮时弹出，**只读确认**（不可编辑），显示当前全局设置摘要：

```
┌─────────────────────────────────┐
│  确认分析                         │
│                                 │
│  压缩    480P / 30fps / 2.00 Mbps │
│  模型    智谱 GLM-4.6V           │
│  音频    多模态解析               │
│                                 │
│  ℹ AI 模型分析可能会产生费用       │
│                                 │
│         [取消]  [开始分析]         │
└─────────────────────────────────┘
```

如需修改设置，需前往设置弹窗。

### 分析进度

时间线样式，阶段数根据音频模式动态变化：

**多模态模式（3 阶段）**：
```
 ● 视频压缩 [====45%====]  ● 视频编码  ● AI 音视频分析
```

**独立 ASR 模式（4 阶段）**：

```
 ● 视频压缩 [====45%====]  ● 视频编码  ● AI 模型分析  ● 语音识别(ASR)
```

- 圆点：等待灰色 → 进行中脉冲动画 + 蓝色 → 完成绿色勾
- 压缩阶段：解析 ffmpeg stderr 推送真实百分比进度条
- 编码阶段：完成时显示压缩后文件大小（自动选择 KB/MB/GB）
- AI 分析阶段子步骤：上传至 AI 服务 → 接收结果 (N 字)
- 独立 ASR 模式：VLM 完成立刻标记，ASR 独立显示"加载语音模型"→"语音识别中"
- 进度条 + 状态文本 + 已用时间
- AI 分析阶段显示打字机效果的随机分析描述（20 条轮播，有子步骤时显示子步骤替代）

### 分析结果

分段列表，每段显示，所有字段可编辑：

```
 ┌───────────────────────────────────────┐
 │ 00:05.20 → 00:12.80  (7.6s)          │
 │ 🖼 一位身穿传统服饰的舞者在...          │  ← 可编辑 (contenteditable)
 │ 💬 对话内容...                         │  ← 可编辑
 │ 📝 字幕内容...                         │  ← 可编辑
 │ [全] [广角] [平视] [固定] [客观]        │  ← 镜头维度（可编辑标签）
 │ [自然] [平静] [自然光] [晴天]           │  ← 场景维度（可编辑标签）
 │ 🔵 白色 ×  🔵 蓝色 ×                  │  ← 颜色（点击 × 移除）
 │ 🏷 舞者 ×  🏷 舞台 ×                  │  ← 主体（点击 × 移除）
 └───────────────────────────────────────┘
```

- 点击分段跳转视频到对应时间
- 播放时高亮当前分段（左边框变主题色）并自动滚动到可见区
- **点击即编辑**：所有文本字段使用 `contenteditable`，点击进入编辑，失焦自动保存
- **标签编辑**：颜色/主体药丸标签有 × 按钮，点击移除并保存
- **编辑态样式**：hover 浅灰背景，focus 主题色边框
- **键盘兼容**：编辑状态时快捷键自动禁用（`isContentEditable` 检测）

## 通用组件

| 组件 | 实现方式 |
|------|---------|
| 通知 | `Quasar.Notify`，顶部弹出，1800ms 自动消失 |
| 对话框 | `q-dialog` + `q-card`，16px 圆角，毛玻璃背景。下拉菜单覆盖方角样式 `.q-menu--square { border-radius: 8px }` |
| 上下文菜单 | 自定义 HTML 定位，含 `q-list` + 快捷键提示 |
| 颜色选择 | 行内圆点，点击切换（非独立组件） |
| 进度条 | `q-linear-progress` |
| 动画 | Lottie（AI 分析等待） |
| 无限滚动 | IntersectionObserver，200px 预加载距离 |
