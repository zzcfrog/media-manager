# 创作工作台 — 设计灵感与思考

> 本文档记录粗剪工作台（Creative Workbench）的设计思考，作为 PRD 设计的基石。

---

## 一、定位

**拍摄后、剪辑前的创意规划工具。**

市场上存在一个空白：
- 素材管理软件（Lightroom / Bridge / Eagle）管理文件，不管理创意
- 剪辑软件（达芬奇 / 剪映 / Premiere）处理技术剪辑，不做创意规划
- 从"我有素材"到"我要讲什么故事"之间，创作者只能靠脑子想、纸笔写

Media Manager 已经完成了第一步——**素材的整理和识别**（AI 分析 16-23 个维度）。创作工作台要做的是第二步——**素材的编排和分镜创意**。

两步完成后，导出 EDL/XML 到剪辑软件进行精剪。

---

## 二、用户画像

### 典型创作者

**旅行 Vlogger（Links photograph / 小鹿Lawrence 类型）**
- 一次旅行拍摄 200-500 条素材，50-100GB
- 没有剧本，"拍完再想故事"（小鹿原话："一通拍，然后根据时间线去组织故事逻辑"）
- 回来面对几百条素材，最大的问题是"不知道从哪里开始"
- 独立完成全流程：拍摄 → 导入 → 选片 → 剪辑 → 调色 → 发布

**纪录片 / 访谈创作者**
- 大量采访素材 + B-roll，需要按主题组织
- 叙事结构复杂，需要反复调整片段顺序
- 对话内容需要精确到秒的定位和筛选

**短视频创作者**
- 从现有素材库中快速出片
- 需要节奏感强的蒙太奇，按音乐节拍剪辑

## 三、核心痛点

### 痛点 1：素材量大，选片如大海捞针

一次旅行回来 300 条素材。创作者需要全部看一遍才能知道有什么，光筛选就要花一整天。我们的 AI 已经做了分析，但分析结果目前只能用来搜索和筛选，没有被组织成创作决策。

### 痛点 2：没有剧本，不知道故事在哪

旅行 vlog 没有剧本。创作者拍完回来，面对 300 条素材要回答一个问题："我要讲什么故事？"这是最难的创意决策。目前只能反复看素材，在脑子里组装。

### 痛点 3：情绪节奏靠直觉，无法可视化

"从平静到高潮再到释然"这种情绪设计，目前完全在创作者脑子里。没有工具能可视化情绪曲线，更不能帮你检查"我的时间线上情绪是不是在正确的地方起落"。

### 痛点 4：剪辑软件太重，不适合做创意探索

达芬奇/剪映是精剪工具，操作门槛高。创作者只是想把几段素材排列组合试试不同叙事，却要面对轨道、特效、调色等复杂界面。创意探索应该是轻量级的。

### 痛点 5：缺乏参照和学习

新手创作者不知道好的视频结构长什么样。没有工具能帮你拆解"这条 Links 的视频为什么好看"——它的情绪节奏、镜头比例、叙事结构是什么。

## 四、页面布局与交互流程

### 整体布局

```
┌─────────────────────────────────────────────────────────────────────┐
│  顶部工具栏：工程名 | 撤销/重做 | AI 策略 | 导出 | 设置           │
├───────────────┬─────────────────────────────────────────────────────┤
│               │  主旨线  ──[开场]────[探索城市]────[回归]──         │
│  素材面板      │  情绪线  ╭╮  ╭─╮ ╭──╮      ╭╮                     │
│               │         ╰╯  ╰─╯ ╰──╯──────╰╯                      │
│  筛选栏        │  旁白线  ──[清晨的...]──[走进...]──[这一刻]──      │
│  搜索框        │  字幕线  ──[ASR...]──[ASR...]──[ASR...]──         │
│               │  文字线  ──"Day1"──"街巷"──"风景"──"回家"──        │
│  segment列表  │  ─────────────────────────────────────────         │
│  [缩略图]标签  │  视频线  ──[seg1]──[seg2]──[seg3]──[seg4]──       │
│  [缩略图]标签  │         ◄──────── 时间轴滚动 ────────►            │
│  [缩略图]标签  ├─────────────────────────────────────────────────────┤
│  ...          │  预览区：选中segment的缩略图/视频预览 + 详情面板    │
│               │  [AI推荐] [替换选项] [segment的16维度标签]          │
├───────────────┴─────────────────────────────────────────────────────┤
│  底部状态栏：总时长 | segment数 | 当前位置 | 缩放滑块              │
└─────────────────────────────────────────────────────────────────────┘
```

### 三栏布局说明

**左栏：素材面板（可折叠，约 280px）**
- 顶部筛选栏（mood / shot_type / scene_type / color_tone 等维度按钮）
- 搜索框（自然语言搜索 segment）
- segment 列表，每项显示：缩略图 + mood 标签 + shot_type + 时长 + 来源视频名
- 列表根据时间线当前选中位置**智能排序**（上下文感知）
- segment 可以展开，显示完整的 16 维度标签和 visual 描述

**中栏：多轨时间线（主体区域）**
- 从上到下：主旨线 → 情绪线 → 旁白线 → 字幕线 → 文字线 → 视频线
- 时间轴水平滚动，支持缩放
- 每条轨道可以折叠/展开
- 视频线上的每个 segment 显示缩略图条 + 简要标签
- 情绪线是可拖拽的曲线，点击锚点可以上下拖动

**右栏/底部：预览与上下文（自适应）**
- 选中 segment 时的视频/图片预览
- 该 segment 的完整分析结果（16 维度标签，可编辑）
- AI 推荐的替代 segment 列表
- 轨道联动的提示信息（"情绪线此处为波峰，但视频线该片段 mood=平静，建议替换"）

### 核心交互流程

**Step 1：创建工程 → 导入素材**

```
点击"新建工程" → 输入工程名和描述
→ 选择要导入的视频/图片（从已有素材库勾选）
→ 确认后，该素材的所有 segment 出现在左侧素材面板
```

**Step 2：搭建叙事结构**

```
方式 A（手动）：
  在主旨线上点击添加章节 → 输入章节标题和描述
  → 从素材面板拖拽 segment 到视频线的对应位置
  → 调整情绪线曲线 → 写旁白和文字

方式 B（AI 辅助）：
  点击"AI 策略" → 选择策略类型（情绪曲线/主题叙事/对话驱动/蒙太奇）
  → 填写主题描述（"京都三日旅行，安静治愈风格"）
  → AI 生成初始故事板，填满所有轨道
  → 用户在此基础上修改
```

**Step 3：调整与替换**

```
点击视频线上的 segment → 右侧显示预览和 AI 推荐替代
→ 不满意？拖拽素材面板的 segment 覆盖替换
→ 或者拖拽情绪线曲线调整情绪 → AI 自动建议匹配的 segment
→ 调整 segment 的起止时间（拖拽边缘）
→ 在旁白线/文字线补充内容
```

**Step 4：预览与导出**

```
点击播放按钮 → 从头到尾按时间线顺序播放所有 segment
（缩略图/低画质快速预览，不是真实渲染）
→ 满意后点击导出 → 选择格式（EDL/XML/故事板PDF/创意简报）
```

### 关键交互细节

**时间线上的 segment 操作**
- 点击选中 → 右侧显示详情和推荐
- 拖拽移动 → 水平拖拽改变顺序
- 拖拽边缘 → 调整 in/out 点（修改起止时间）
- 右键菜单 → AI 替换 / 删除 / 复制 / 在此处插入空位

**情绪线曲线操作**
- 默认由 AI 根据视频线 segment 的 mood 自动生成
- 点击曲线上任意位置添加锚点 → 上下拖拽调整情绪值
- 拖拽后 AI 检查视频线对应 segment 是否匹配，不匹配则高亮提示
- 双击锚点删除

**轨道联动提示**
- 情绪线与视频线不匹配时，情绪线上显示黄色/红色标记
- 旁白线有内容但视频线为空时，视频线显示虚线占位
- AI 自动推荐出现在右侧面板，不弹窗打断操作

## 五、多轨创意时间线

传统剪辑软件是**单维度时间线**（视频轨道 + 音频轨道）。创作工作台是**多维创意时间线**——每条轨道代表一个创作维度：

```
主旨线  ──── [旅行开始] ──── [探索城市] ──── [发现自我] ──── [回归平静] ────
情绪线  ╭─╮     ╭──╮  ╭───╮         ╭─╮    ╭─╮
        │ │  ╭─╮│  │╭─│   │╮   ╭──╮ │ │ ╭╮ │ │
        ╰─╯  │ ╰╯  ╰╯ ╰   ╯╰───╯  ╰─╯ ╰╯ ╰─╯
         ↑平静  ↑好奇   ↑兴奋   ↑壮丽  ↑温暖
旁白线  ──── [清晨的京都...] ──── [走进小巷...] ──── [这一刻...] ────
字幕线  ──── [ASR字幕...] ──── [ASR字幕...] ──── [ASR字幕...] ────
文字线  ──── "Day 1" ──── "迷失在街巷" ──── "最好的风景" ──── "回家"
视频线  ──── [片段1] ──── [片段2] ──── [片段3] ──── [片段4] ──── [片段5] ────
         ←─── 调整每段起止时间 ───→
```

### 轨道设计意图

**视频线（主时间轴）**— 排列选中的片段，可调整每段的 in/out 点。这是最终导出的核心。

**情绪线（曲线形式）** — 连续的情绪曲线，垂直于时间线方向：
- 波峰 = 高涨情绪（兴奋、壮丽、紧张、欢快）
- 波谷 = 低落情绪（忧伤、孤独、恐怖）
- 中间值 = 平静（平静、温馨、浪漫）
- 曲线可手动拖拽调整，也可由 AI 自动从片段的 `mood` 字段生成
- 拖拽曲线时，AI 实时建议匹配当前情绪值的片段

情绪值的映射规则：每个 mood 枚举映射到一个数值范围（如 壮丽→0.9, 平静→0.5, 忧伤→0.1），曲线在片段之间做平滑插值。

**旁白线** — 写入旁白文本。这是很多 vlog 创作者的实际工作方式：先有旁白思路，再配画面。AI 可以根据旁白内容推荐匹配的片段。

**字幕线** — 显示视频片段中 ASR 识别出的语音字幕。可以作为剪辑时的对话/旁白参考，也可以直接导出为字幕文件。

**文字线** — 屏幕文字、标题卡、字幕提示。用于标记章节和视觉节奏点。

**主旨线** — 视频的叙事结构。把整个视频分成几个章节/段落，每个段落有一个主题。这决定了视频的骨架。

### 轨道间的联动（核心亮点）

轨道不是独立的，而是互相约束、互相建议：

- 在**旁白线**写"清晨，街道还很安静" → AI 从素材库推荐匹配的片段填入视频线对应位置
- 拖拽**情绪线**曲线到波峰 → AI 检查视频线对应片段的 mood/shot_type/camera_movement 是否匹配，不匹配则建议替换
- 在**主旨线**写"从喧嚣到宁静" → AI 按照这个主题方向筛选素材
- 替换视频线的某段 → AI 自动更新情绪线的曲线数值和字幕线内容
- 调整情绪线的曲线 → AI 建议替换不匹配的视频片段
- 拖入新片段到视频线 → 字幕线自动填充该片段的 ASR 内容

这种联动是传统剪辑软件完全没有的能力。

## 六、AI 剪辑策略（可混合使用）

同一条时间线上可以组合多种策略。比如一条旅行视频：

```
主旨线  [开场钩子] ─────── [正片：京都三日] ─────── [结尾：回归]
策略     节奏蒙太奇        主题叙事驱动            情绪曲线
时长     15秒              4分钟                   30秒
```

不同章节可以用不同策略生成，手动编排也可以和 AI 策略混用。

多轨时间线是骨架，AI 策略是血肉。不同类型的视频需要不同的策略：

### 策略 1：情绪曲线驱动

适合：旅行 vlog、风光片

用户画出期望的情绪曲线，AI：
1. 从素材库中筛选 mood 匹配的片段
2. 按"平静→兴奋→高潮→回归"的节奏排列
3. 确保视觉连续性（相邻片段的 scene_type / color_tone / lighting 不冲突）
4. 自动标注主旨线的章节划分

### 策略 2：主题叙事驱动

适合：纪实、专题、旅行日记

用户给一个主题描述（"京都三日，从游客到旅人"），AI：
1. 分析所有素材的 visual / main_subjects / scene_type
2. 识别与主题相关的片段群
3. 提出几种叙事结构（按时间线 / 按空间 / 按情绪递进 / 按对比）
4. 每种结构生成一个完整的故事板供用户选择

### 策略 3：对话驱动

适合：访谈、纪录片、有大量 ASR 内容的视频

AI：
1. 提取所有对话/旁白片段并做内容摘要
2. 用户编写叙事大纲（或 AI 根据对话生成）
3. AI 把对话片段匹配到大纲的每个章节
4. 为每个说话段落推荐 B-roll（根据对话内容的 visual 描述匹配）
5. 自动生成旁白线的草稿文本

### 策略 4：节奏蒙太奇

适合：短视频、社交媒体内容

用户选一批素材，指定时长（30s/60s/3min），AI：
1. 按视觉节奏（shot_type 交替、camera_movement 变化）选择切点
2. 确保 color_tone 和 lighting 的视觉连贯性
3. 如果有背景音乐，按节拍卡切点
4. 自动生成快节奏序列

## 七、功能灵感

### 素材缺口检测

叙事结构搭好后，AI 检查时间线并报告：

- "情绪线在位置 3-4 标注了'从平静到兴奋'的过渡，但视频线没有合适的过渡片段"
- "你连续用了 4 个室内镜头，建议加一个室外建立镜头"
- "第 2 段和第 3 段的色彩从暖调跳到冷调，如果是有意的设计可以忽略"
- "旁白线第 2 章提到'海边'，但视频线没有海洋相关的片段"

这让创作者在进入剪辑之前就知道还缺什么。

### 参考视频解构

用户给一个喜欢的参考视频，AI 分析它的结构：
- 镜头类型分布（全景 vs 特写 vs 中景的比例）
- 情绪曲线（哪里平静、哪里高潮、持续多久）
- 节奏（平均镜头时长、切点频率）
- 色彩走向（色彩温度如何随时间变化）

然后提供一个同结构的"骨架"，用用户自己的素材去填充。对新手创作者尤其有价值。

### AI 片段替换

在时间线上点击任意片段，AI 从素材库推荐相似但不同的替代选项：
- 相同场景、不同角度
- 相同情绪、不同内容
- 相同色彩、不同构图
- 按与当前片段的相似度排序

创作者可以快速比较和切换，找到最合适的那个。

### 智能修剪

每段素材有 time_start / time_end，AI 可以建议最佳起止点：
- 对话场景：自动切到说话内容的核心段落
- 情绪转折：保留情绪最饱满的几秒
- 风光镜头：保留最佳光线条件的部分
- 运动镜头：保留运镜最流畅的部分

### 版本分支

创作是探索性的。用户应该能：
- 保存当前时间线为一个版本
- 在新版本上大胆尝试不同排列
- 随时对比和切换版本
- 从任意版本继续衍生

### 叙事模板

预设几种经典叙事结构，用户选择后 AI 按模板组织素材：

- **三幕剧**：建置 → 冲突 → 解决
- **英雄之旅**：出发 → 试炼 → 转变 → 回归
- **一天的时间**：清晨 → 正午 → 黄昏 → 夜晚
- **情绪弧线**：平静 → 积累 → 高潮 → 释放
- **对比叙事**：喧嚣/宁静、城市/自然、白天/夜晚交替

## 八、可利用的数据资产

创作工作台不需要从零开始——我们的 AI 分析已经提供了丰富的结构化数据：

### 已有分析维度（每个视频片段）

| 维度 | 创作用途 |
|------|----------|
| `visual`（画面描述） | 自然语言搜索、主题匹配 |
| `asr`（语音内容） | 对话驱动策略、旁白草稿 |
| `mood`（情绪） | 情绪曲线、情绪匹配 |
| `shot_type`（景别） | 镜头节奏、景别比例分析 |
| `camera_movement`（运镜） | 动态节奏、视觉连贯性 |
| `scene_type`（场景） | 场景分类、空间叙事 |
| `color_tone`（色调） | 视觉连贯性、色彩走向 |
| `lighting`（光线） | 时间推断、氛围匹配 |
| `weather`（天气/时段） | 时间线叙事（清晨→夜晚） |
| `dominant_colors`（主色） | 色彩主题、视觉节奏 |
| `main_subjects`（主体） | 主题聚类、叙事线索 |
| `style`（风格） | 风格一致性检查 |
| `composition`（构图） | 视觉质量评估 |

### 可补充的数据

- **每段片段的视觉 embedding**（目前只有整张图片，可以扩展到视频关键帧）
- **音频特征**（节奏、音量、安静/嘈杂）用于辅助节奏编辑
- **镜头运动强度**（从 camera_movement + 时间长度推算节奏感）
- **色彩分布曲线**（每段的色彩温度变化，用于检查视觉连贯性）

## 九、导出与衔接

创作工作台不是终点，而是剪辑的起点。导出格式：

- **EDL（Edit Decision List）** — 通用剪辑列表，达芬奇/Premiere/FCP 都支持
- **XML（FCPXML / Premiere XML）** — 保留更多时间线信息
- **故事板 PDF** — 每个片段的缩略图 + 起止时间 + 旁白文字，给剪辑师参考
- **创意简报** — 整个视频的创作意图、章节结构、情绪设计，作为剪辑师的理解文档

## 十一、差异化优势

为什么创作者不用剪辑软件直接做这些？

1. **有结构化 AI 数据** — 剪辑软件不知道你的片段是"壮丽"还是"忧伤"，我们知道
2. **轨道间联动** — 剪辑软件的轨道是独立的，我们的轨道互相建议
3. **轻量级** — 不需要处理渲染、特效、调色，专注于创意规划
4. **AI 策略** — 从"大海捞针"到"AI 帮你捞"，再到现在"AI 帮你摆盘"
5. **已有的素材库** — 不需要重新导入，分析数据就在那里

## 十二、素材交互：拖拽、替换与推荐

时间线的左侧是**素材面板**，展示当前工程中导入的所有候选素材（可按维度筛选）。

**关键概念：操作单位是 segment，不是 video。** 一个视频经过 AI 分析后会被分成多个 segment，每个 segment 有独立的时间范围、画面描述、情绪、镜头语言等。素材面板展示和拖拽的最小单位是 segment。用户也可以展开查看 segment 所属的原始视频。

### 素材面板结构

```
素材面板
├── 筛选栏：按 mood / shot_type / scene_type / color_tone 筛选
├── 搜索框：自然语言搜索 segment 的 visual / asr 内容
└── 片段列表（按上下文智能排序）
    ├── [缩略图] 广角全景·海边·平静  00:12-00:35
    │           来自 DJI_20260401_0002_D.MP4
    ├── [缩略图] 中景·街道·温馨     00:45-01:02
    │           来自 GX100_0401_0012.MP4
    └── ...
```

### 拖入素材到时间线

**基础方式：从素材面板拖拽 segment**
- 拖拽 segment 到视频线的空位或两段之间插入
- 拖拽时高亮目标位置，显示 segment 时长
- 插入后视频线显示该 segment 的起止时间（来自原始视频的 time_start / time_end）

**快捷方式：双击素材面板的 segment**
- 双击自动追加到视频线末尾
- 按住 Shift 双击插入到当前选中位置之后

### 替换已有 segment

**点击替换**
- 点击时间线上的 segment → 弹出替换面板，AI 推荐 3-5 个替代 segment
- 推荐逻辑：相同场景 + 不同角度 / 相同情绪 + 不同内容 / 相同色彩 + 不同构图
- 每个推荐显示来源视频和 segment 时间范围
- 按相似度排序，顶部是最接近的替代

**拖拽替换**
- 从素材面板拖拽 segment 到已有 segment 上方 → 原 segment 高亮 → 松手替换
- 替换时自动保持相同的时长区间

**AI 一键替换**
- 选中时间线上的 segment → 右键"AI 替换" → AI 根据上下文（前后 segment 的情绪、色彩、场景）自动选一个最合适的替代

### 素材面板的智能排序

素材面板不只是静态列表。根据用户当前操作的位置，面板自动重排序：
- 时间线选中的是"海边全景" segment → 素材面板把其他海边/水景 segment 排到前面
- 情绪线在当前位置是波峰 → 素材面板优先显示 mood=壮丽/兴奋 的 segment
- 旁白线写了"清晨" → 素材面板优先显示 weather=晨昏 的 segment

这种**上下文感知排序**让创作者在找素材时几乎不需要主动搜索。

### 空位占位

时间线上可以预留空位（ghost slot），标注"这里需要一段XX的素材"。AI 会持续提示匹配的候选片段。创作者可以先搭结构再填素材。

## 十三、风险与挑战

1. **AI 策略的质量** — 排序建议如果不好用，创作者会直接忽略。需要先做 MVP 验证
2. **学习成本** — 多轨时间线是新的交互模式，需要引导和模板降低门槛
3. **导出兼容性** — EDL/XML 格式各剪辑软件支持程度不同，需要测试
4. **素材精度** — 时间线以 segment 为操作单位（几秒到几十秒），导出时携带原始视频路径 + segment 起止时间，剪辑软件可进一步精调
5. **离线使用** — AI 策略调用 LLM API，离线场景下只能使用手动模式
