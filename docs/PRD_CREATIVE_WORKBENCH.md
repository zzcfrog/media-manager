# PRD — 创作工作台（Creative Workbench）

## 1. 产品定位

创作工作台是 Media Manager 的核心功能扩展，定位为**拍摄后、剪辑前的创意规划工具**。

它解决的核心问题是：创作者拥有大量已分析的素材，但缺乏工具将这些素材组织成有叙事结构的视频方案。创作工作台填补了素材管理和视频剪辑之间的空白——不处理渲染、特效、调色，专注于**素材的编排和分镜创意**。

最终产出为 EDL/XML 等格式，导出至达芬奇、Premiere、剪映等专业剪辑软件进行精剪。

## 2. 目标用户

| 用户类型 | 典型场景 | 核心需求 |
|----------|----------|----------|
| 旅行 Vlogger | 一次旅行 200-500 条素材，无剧本，回来后组织故事 | 从海量素材中发现叙事线索 |
| 纪录片/访谈创作者 | 大量采访 + B-roll，复杂叙事结构 | 按主题组织对话，匹配 B-roll |
| 短视频创作者 | 从现有素材库快速出片 | 节奏感强的蒙太奇，快速生成 |

## 3. 核心概念

### 3.1 工程（Project）

工程是创作工作台的最小工作单元。每个工程对应一个视频项目。

**工程属性**：
- 工程名称
- 创作描述（如"60秒京都旅行vlog，安静治愈风格"）
- 关联的素材（从素材库勾选已分析的视频/图片）
- 多条时间线版本（支持分支探索）

**工程与素材库的关系**：
- 工程引用素材库中的 segment，不复制原始文件
- 同一个素材可以被多个工程使用
- 素材库中删除素材时，工程中引用该素材的 segment 标记为"素材已缺失"

### 3.2 Segment 为操作单位

**操作单位是 segment，不是 video。** 一个 5 分钟的视频经 AI 分析后被分成 10-20 个 segment，每个 segment 有独立的起止时间、画面描述、情绪、镜头语言等 16+ 维度标签。

素材面板展示和拖拽的最小单位是 segment。用户也可以展开查看 segment 所属的原始视频。

### 3.3 多轨创意时间线

时间线由 6 条轨道组成，从上到下：

```
主旨线  ──── [旅行开始] ──── [探索城市] ──── [发现自我] ──── [回归平静] ────
情绪线  ╭─╮     ╭──╮  ╭───╮         ╭─╮    ╭─╮
        │ │  ╭─╮│  │╭─│   │╮   ╭──╮ │ │ ╭╮ │ │
        ╰─╯  │ ╰╯  ╰╯ ╰   ╯╰───╯  ╰─╯ ╰╯ ╰─╯
         ↑平静  ↑好奇   ↑兴奋   ↑壮丽  ↑温暖
旁白线  ──── [清晨的京都...] ──── [走进小巷...] ──── [这一刻...] ────
字幕线  ──── [ASR字幕...] ──── [ASR字幕...] ──── [ASR字幕...] ────
文字线  ──── "Day 1" ──── "迷失在街巷" ──── "最好的风景" ──── "回家"
视频线  ──── [seg1] ──── [seg2] ──── [seg3] ──── [seg4] ──── [seg5] ────
```

#### 主旨线

视频的叙事结构骨架。把整个视频分成若干章节，每个章节有标题和描述。主旨线决定了视频的段落划分。

- 点击添加章节 → 输入标题和描述
- 章节可拖拽调整顺序
- 每个章节可指定不同的 AI 策略

#### 情绪线（曲线形式）

连续的情绪曲线，垂直于时间线方向。波峰 = 高涨情绪，波谷 = 低落情绪，中间 = 平静。

**情绪值映射**：

| 情绪 | 数值 |
|------|------|
| 壮丽 / 兴奋 / 紧张 / 欢快 | 0.7 - 1.0 |
| 浪漫 / 温馨 / 梦幻 | 0.4 - 0.7 |
| 平静 / 神秘 / 怀旧 | 0.2 - 0.5 |
| 忧伤 / 孤独 / 恐怖 | 0.0 - 0.3 |

**交互方式**：
- AI 根据视频线 segment 的 mood 字段自动生成初始曲线
- 点击曲线上任意位置添加锚点，上下拖拽调整情绪值
- 拖拽后 AI 检查视频线对应 segment 是否匹配，不匹配则高亮提示
- 双击锚点删除

#### 旁白线

旁白/解说文本。每段旁白对应时间线上的一个位置区间。

- 手动输入旁白文字
- AI 根据旁白内容推荐匹配的 segment
- 旁白线的内容作为导出时的 SRT 字幕和创意简报的一部分

#### 字幕线

显示视频 segment 中的 ASR 语音识别结果。

- 拖入 segment 时自动填充该 segment 的 asr 字段内容
- 可手动编辑
- 导出时可生成 SRT 字幕文件

#### 文字线

屏幕文字、标题卡、章节标题。

- 每段文字独立添加和编辑
- 用于标记视觉节奏点（如"Day 1"、"第三章"）
- 导出时作为标题叠加层的参考

#### 视频线（主时间轴）

排列选中的 segment，是最终导出的核心。

- 每个 segment 显示为缩略图条 + 简要标签（mood + shot_type）
- 可调整每段的 in/out 点（拖拽边缘修改起止时间）
- 拖拽移动改变顺序
- 支持空位占位（标注"需要XX类型素材"）

#### 轨道间联动

| 用户操作 | AI 联动响应 |
|----------|-------------|
| 旁白线写入"清晨，街道安静" | 推荐匹配 segment 填入视频线 |
| 情绪线曲线拖到波峰 | 检查视频线 mood 是否匹配，不匹配建议替换 |
| 主旨线写入"从喧嚣到宁静" | 按主题方向筛选素材 |
| 替换视频线 segment | 自动更新情绪线曲线 + 字幕线内容 |
| 拖入新 segment 到视频线 | 字幕线自动填充 ASR 内容 |
| 情绪线与视频线不匹配 | 情绪线显示黄色/红色标记 |

## 4. 导航入口

在左侧边栏现有"浏览"区域（素材库、我的喜欢、查找相似…）之下，新增独立区域"编排"，与浏览平级。

```
┌─────────────┐
│  浏览        │
│    素材库     │
│    我的喜欢   │
│    查找相似   │
├─────────────┤
│  编排        │
│    工程 A    │
│    工程 B    │
│  + 新建工程   │
└─────────────┘
```

- "编排"区域列出所有已创建的工程，点击进入对应工程的工作台页面
- 底部"+ 新建工程"按钮 → 弹窗输入名称和描述 → 创建后跳转工作台
- 工程列表按最近更新时间排序
- 右键工程可重命名或删除

└─────────────────────────────────────────────────────────────────────┘

### 左栏：素材面板（可折叠，约 280px）

- 筛选栏：按 mood / shot_type / scene_type / color_tone 等维度筛选
- 搜索框：自然语言搜索 segment 的 visual / asr 内容
- segment 列表：每项显示缩略图 + mood 标签 + shot_type + 时长 + 来源视频名
- 上下文智能排序：根据时间线当前选中位置自动重排
- segment 可展开显示完整 16 维度标签

### 中栏：多轨时间线（主体区域）

- 六条轨道，每条可折叠/展开
- 时间轴水平滚动，支持缩放
- 视频线上每个 segment 显示缩略图条 + 简要标签
- 情绪线为可拖拽曲线

### 右栏/底部：预览与上下文（自适应）

- 选中 segment 的视频/图片预览
- 完整分析结果（16 维度标签，可编辑）
- AI 推荐的替代 segment 列表
- 轨道联动提示

## 6. 用户流程

### Step 1：创建工程

1. 点击"新建工程"
2. 输入工程名称和创作描述
3. 从素材库勾选要导入的视频/图片
4. 确认后，所选素材的所有已分析 segment 出现在左侧素材面板

### Step 2：搭建叙事结构

**方式 A — 手动编排**：
1. 在主旨线上添加章节，输入标题
2. 从素材面板拖拽 segment 到视频线对应位置
3. 调整情绪线曲线
4. 在旁白线/文字线补充内容

**方式 B — AI 策略生成**：
1. 点击顶部"AI 策略"
2. 选择策略类型，填写参数
3. AI 生成初始故事板，填满所有轨道
4. 用户在此基础上修改

**方式 C — 混合模式**：
1. 用 AI 策略生成某个章节的初始排列
2. 手动拖拽调整细节
3. 其他章节用不同策略或纯手动编排

### Step 3：调整与替换

1. 点击视频线 segment → 右侧显示预览和 AI 替代推荐
2. 拖拽素材面板 segment 覆盖替换不满意的片段
3. 拖拽情绪线曲线 → AI 建议匹配 segment
4. 拖拽 segment 边缘调整 in/out 点
5. 在各轨道补充文字内容

### Step 4：预览与导出

1. 点击播放 → 按时间线顺序播放所有 segment（缩略图/低画质预览）
2. 满意后点击导出 → 选择格式
3. 导出文件可导入剪辑软件继续精剪

## 7. AI 剪辑策略

同一条时间线上可以组合多种策略，不同章节使用不同策略：

```
主旨线  [开场钩子] ─────── [正片：京都三日] ─────── [结尾：回归]
策略     节奏蒙太奇        主题叙事驱动            情绪曲线
时长     15秒              4分钟                   30秒
```

### 6.1 情绪曲线驱动

**适合**：旅行 vlog、风光片

**输入**：用户在情绪线上画出期望曲线（或描述目标情绪走向）

**AI 处理**：
1. 从素材库筛选 mood 匹配的 segment
2. 按情绪节奏排列（平静→兴奋→高潮→回归）
3. 检查视觉连续性（相邻 segment 的 scene_type / color_tone / lighting）
4. 自动标注主旨线的章节划分
5. 填充字幕线和旁白线草稿

### 6.2 主题叙事驱动

**适合**：纪实、专题、旅行日记

**输入**：主题描述（如"京都三日，从游客到旅人"）+ 预期时长

**AI 处理**：
1. 分析所有 segment 的 visual / main_subjects / scene_type
2. 识别与主题相关的 segment 群
3. 提出多种叙事结构（按时间 / 按空间 / 按情绪递进 / 按对比）
4. 每种结构生成一个完整故事板供用户选择

### 6.3 对话驱动

**适合**：访谈、纪录片、有大量 ASR 内容的视频

**输入**：叙事大纲（手动编写或 AI 根据对话内容生成）

**AI 处理**：
1. 提取所有对话/旁白 segment 并做内容摘要
2. 将对话 segment 匹配到大纲的每个章节
3. 为每个说话段落推荐 B-roll（根据对话内容的 visual 描述匹配）
4. 自动生成旁白线草稿

### 6.4 节奏蒙太奇

**适合**：短视频、社交媒体内容

**输入**：目标时长（30s/60s/3min）+ 风格描述

**AI 处理**：
1. 按 visual 节奏（shot_type 交替、camera_movement 变化）选择切点
2. 确保 color_tone 和 lighting 的视觉连贯性
3. 自动生成快节奏序列
4. 可选：提供背景音乐，AI 按节拍卡切点

## 8. 素材交互

### 7.1 拖入 segment

- 从素材面板拖拽 segment 到视频线空位或两段之间插入
- 拖拽时高亮目标位置，显示 segment 时长
- 双击 segment 追加到视频线末尾
- Shift + 双击插入到当前选中位置之后

### 7.2 替换 segment

**点击替换**：
- 点击时间线 segment → 右侧显示 AI 推荐的 3-5 个替代 segment
- 推荐逻辑：相同场景 + 不同角度 / 相同情绪 + 不同内容 / 相同色彩 + 不同构图
- 按相似度排序

**拖拽替换**：
- 从素材面板拖拽 segment 到已有 segment 上方 → 原 segment 高亮 → 松手替换
- 替换时自动保持相同时长区间

**AI 一键替换**：
- 右键"AI 替换" → AI 根据上下文自动选择最合适的替代

### 7.3 上下文智能排序

素材面板根据当前操作位置自动重排序：
- 时间线选中"海边全景" → 素材面板优先显示海边/水景 segment
- 情绪线在波峰位置 → 优先显示 mood=壮丽/兴奋 的 segment
- 旁白线写了"清晨" → 优先显示 weather=晨昏 的 segment

### 7.4 空位占位

时间线上可预留空位（ghost slot），标注"需要XX类型素材"。AI 持续提示匹配候选。创作者可以先搭结构再填素材。

## 9. 预览与导出

### 预览

点击播放按钮，按时间线顺序快速播放所有 segment（缩略图序列或低画质预览，非真实渲染）。用于检查叙事节奏和情绪走向。

### 导出格式

| 格式 | 说明 | 用途 |
|------|------|------|
| EDL | Edit Decision List，通用剪辑列表 | 达芬奇 / Premiere / FCP 通用导入 |
| XML | FCPXML / Premiere XML | 保留更多时间线和素材信息 |
| 故事板 PDF | 缩略图 + 起止时间 + 旁白文字 | 给剪辑师或团队参考 |
| 创意简报 | 创作意图 + 章节结构 + 情绪设计 + 主旨描述 | 与剪辑师沟通的创作文档 |
| SRT | 字幕文件 | 旁白线 + 字幕线导出为标准字幕 |

## 11. 演进路径

```
Phase 1（当前）                     Phase 2                          Phase 3
素材管理 + AI 分析          →    创作工作台 MVP              →    智能创作引擎

· 导入/浏览/筛选               · 工程管理                      · AI 自动生成故事板
· AI 分析 16+ 维度             · 多轨时间线（手动）             · 参考视频解构
· 全文搜索                     · segment 拖拽/替换/调整         · 版本分支与对比
· 标签/评分/颜色标签           · 基础情绪线（手动标注）         · 素材缺口检测
· XMP 写入                     · EDL 导出                      · 叙事模板
                               · 素材面板 + 智能排序             · AI 片段替换建议
                                                               · 对话/旁白驱动编辑
```

## 12. 限制与风险

| 风险 | 影响 | 应对 |
|------|------|------|
| AI 排序建议质量不稳定 | 用户忽略 AI 建议，退化为纯手动 | Phase 2 先做手动模式，AI 作为辅助 |
| 多轨时间线学习成本高 | 新用户不知如何开始 | 提供叙事模板和引导流程 |
| EDL/XML 导出兼容性 | 不同剪辑软件支持程度不同 | 优先支持 EDL（最通用），逐步扩展 |
| 离线无法使用 AI 策略 | 无网络时只能手动 | 离线模式隐藏 AI 策略按钮 |
| segment 粒度可能不够细 | 几秒到几十秒的 segment，剪辑需帧级 | 导出时携带原始视频路径 + 时间码，剪辑软件可精调 |
