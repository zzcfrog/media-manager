# Media Manager 使用指南

## 快速开始

### 环境要求

- Python 3.12+
- [ffmpeg](https://ffmpeg.org/)（视频压缩、缩略图生成、实时转码）
- [exiftool](https://exiftool.org/)（相机元数据、XMP 写入）

### 安装

```bash
pip install -r requirements.txt
```

### 启动

**方式一：浏览器**（开发调试）

```bash
python3 run.py
```

启动后自动打开浏览器访问 `http://127.0.0.1:6622`。

**方式二：Electron 桌面端**

```bash
cd electron && npm start
```

自动启动后端 Python 进程，窗口大小 1600×900，最小 1000×600。

## 导入媒体

点击左下角"导入"按钮打开导入弹窗，支持两种选择方式：

- **选择文件**：弹出系统文件选择器（支持多选）
- **选择目录**：选择整个文件夹，递归扫描
- 也可以直接将文件或文件夹拖拽到弹窗区域

导入流程自动执行：扫描匹配文件 → 3 个并发导入 → 生成缩略图 → 提取元数据。已导入的重复文件会自动跳过。

> 导入只是将文件信息写入数据库，不会移动或复制原始文件。

## 浏览与筛选

### 网格视图与列表视图

通过筛选栏右侧的视图切换按钮在两种视图间切换，也可以按 `G` 键快速切换。

- **网格视图**：缩略图卡片，底部滑块控制卡片大小（0.5x ~ 2x）
- **列表视图**：表头固定、可排序的表格形式

### 筛选栏

| 筛选项 | 说明 |
|--------|------|
| 类型切换 | 全部 / 图片 / 视频 |
| 星级筛选 | 点击 1-5 星筛选最低评分 |
| 颜色标签 | 红、黄、绿、蓝、紫五色筛选 |
| 喜欢 | 三态切换：全部 → 已喜欢 → 未喜欢（斜杠标识"排除"） |
| 分析状态 | 三态切换：全部 → 已分析 → 未分析（斜杠标识"排除"） |
| 搜索框 | 全文搜索（文件名 + 分析内容） |
| 排序 | 按导入时间/拍摄时间/名称/分辨率/时长/大小/评分排序 |
| 分组 | 按时间（日/周/月/季/年）或按时长分组显示 |

所有筛选条件、排序、视图模式、文件夹选择自动保存，刷新页面后保持。

> Footer 居中显示当前所有活跃的筛选条件标签，无筛选条件时不显示。

### 选择操作

- **单击**：选中
- **Ctrl/Cmd + 点击**：切换选中
- **Shift + 点击**：范围选中
- **拖拽框选**：在空白区域按住鼠标拖拽，自动框选
- **右键菜单**：查看详情、移除、查找相似、AI 分析、写入 XMP

## 媒体详情

双击画廊中的卡片或选中后按 `Enter` 进入详情页。

### 视频详情

- **元数据栏**：视频信息（分辨率/时长/编码/帧率/码率/色彩空间/色彩曲线）、音频信息、拍摄设备、文件信息
- **视频播放器**：原生 HTML5 播放控件
- **音频波形**：点击波形跳转到对应时间点
- **视频示波器**：波形图（Waveform）、RGB 三通道（Parade）、色度图（Vectorscope），播放时 ~15fps 实时渲染

### 图片详情

- **元数据栏**：图像信息（分辨率/色彩空间/位深）、拍摄设备、文件信息
- **图片预览**：支持缩放（鼠标滚轮/触控板张合）和拖拽平移，缩放范围 25%-500%
- **RGB 直方图**：三通道色彩分布

### 工具栏

返回按钮 | 文件名 | 星级评分 | 颜色标签 | 收藏 | XMP 写入（仅图片）

## AI 智能分析

### 开始分析

**单条分析**：在详情页右侧面板点击"AI 分析"按钮，弹出确认对话框显示当前设置摘要（压缩参数、AI 模型、预估码率），确认后开始分析。

**批量分析**：在画廊中选中一个或多个素材，右键选择"AI 分析"，弹出确认对话框显示：
- 素材总数（视频/图片分别计数）
- 视频/图片分析模型
- 已分析素材数量及处理策略（重新分析 / 跳过）
- 确认后在顶部进度条跟踪所有任务，已提交的素材自动跳过

### 分析流程

**视频分析**（默认多模态模式，3 个阶段）：

1. **视频压缩** — ffmpeg 压缩到设定分辨率/帧率，实时进度条
2. **视频编码** — 完成后显示压缩文件大小
3. **AI 综合分析** — 上传至 AI 服务 → 接收分析结果

**图片分析**（单阶段）：

- AI 图像分析 — 上传至 AI 服务 → 接收分析结果

### 分析结果

分析完成后，右侧面板显示分段列表。每段包含：

- **时间范围**：点击跳转视频到对应时间
- **视觉描述**：AI 对画面内容的描述
- **对话内容**：识别出的语音文字
- **字幕内容**：识别出的字幕
- **镜头维度**：景别、焦段、视角、运镜、透视
- **场景维度**：场景类型、情绪、光线、天气
- **主体/颜色**：识别出的主要拍摄对象和主色调

所有字段均可直接点击编辑，失焦自动保存。

### 删除分段

点击分段右上角的删除按钮，可选择三种删除方式：

- 延长前一段的结束时间
- 延长后一段的开始时间
- 直接删除

## 收藏与评分

### 评分

在画廊或详情页中，点击星级 1-5 星评分。点击已选中的星级取消评分。也可以在画廊中选中多个文件后按 `0-5` 批量评分。

### 收藏

点击红心按钮切换收藏状态，选中多文件后按 `L` 批量切换。

### 颜色标签

五色圆点（红、黄、绿、蓝、紫），点击切换，用于分类标记。

---

## 合集与标签

### 合集

- 在侧边栏"合集"区域点击 `+` 创建合集
- 在画廊中选中文件后，右键菜单添加到合集
- 点击侧边栏中的合集名称筛选显示

### 标签

- 在画廊中选中文件后，右键菜单或详情页添加标签
- 侧边栏显示所有标签及使用次数
- 点击标签名筛选画廊
- AI 分析完成后，分析结果中的主体、颜色、场景等会自动关联为标签

---

## 查找重复

在侧边栏点击"查找重复"进入，支持两种模式：

- **精确重复**：基于 SHA-256 文件哈希，找出完全相同的文件
- **相似图片**：基于感知哈希（pHash），找出视觉相似的图片

可点击"计算哈希"补全缺失的哈希值。

---

## XMP 侧车文件

图片完成 AI 分析后，可以点击工具栏中的 XMP 按钮将分析结果写入 XMP 侧车文件（`.xmp`）。写入内容包括：

- 评分、颜色标签
- AI 分析标签（主体、颜色、场景等）
- AI 描述文本

写入的 XMP 数据兼容 Adobe Lightroom / Bridge 等软件。支持在画廊中选中多张图片批量写入。

## 设置

点击顶部工具栏的齿轮图标打开设置弹窗。

### 图片分析

| 设置项 | 选项 |
|--------|------|
| 压缩尺寸 | 1920（推荐）/ 1280 / 800 / 原图 |
| AI 模型 | GLM-4V-Plus / GLM-4.6V / GLM-4.6V-Flash / GLM-4.6V-FlashX / GLM-4.5V |
| API Key | 图片分析独立密钥，必须设置 |

### 视频分析

| 设置项 | 选项 |
|--------|------|
| 压缩分辨率 | 480P（推荐）/ 320P / 240P |
| 帧率 | 30fps（推荐）/ 20fps / 10fps |
| 预估码率 | 根据分辨率和帧率自动计算并显示 |
| 硬件加速 | 开关（仅在检测到硬件编码器时显示，如 macOS VideoToolbox） |
| AI 模型 | 与图片分析相同的模型列表 |
| API Key | 视频分析独立密钥 |

### 音频分析

| 设置项 | 选项 |
|--------|------|
| 多模态解析 | 默认开启，AI 模型同时处理视觉和语音 |
| ASR 引擎 | 关闭多模态后可选：本地 Whisper / 智谱 ASR |
| ASR API Key | 云端 ASR 时填写，本地模型可留空 |

## 支持的格式

### 视频

MP4, MOV, AVI, MKV, WebM, M4V, FLV, WMV, 3GP, MTS, M2TS

> 非 H.264 格式（AVI、MKV、MTS 等）播放时自动转码。

### 图片

**常见格式**：JPEG, PNG, WebP, BMP, TIFF

**Apple/HEIF**：HEIC, HEIF, HIF（需 pillow-heif 解码，分析时自动压缩）

**AVIF**

**RAW 格式**（40+ 种）：
Adobe DNG · Canon CR2/CR3/CRW · Nikon NEF/NRW · Sony ARW/SRF/SR2 · Fujifilm RAF · Olympus ORF · Panasonic/Leica RAW/RW2/RWL · Hasselblad 3FR/FFF · Phase One IIQ · Pentax PEF · Sigma X3F · Samsung SRW · Mamiya/Leaf MEF/MOS · Kodak KDC/DCR · Minolta MRW · Apple ProRAW

## 数据存储

所有数据存储在 `data/` 目录下：

- `media.db` — SQLite 数据库（媒体信息、分析结果、合集、标签）
- `thumbnails/` — 生成的缩略图

可通过 `MEDIA_MANAGER_HOME` 环境变量更改数据目录。
